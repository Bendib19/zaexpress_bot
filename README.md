# ZExpress Telegram Bot

## كيفاش تستعمل المشروع:

1. انسخ `.env.example` وسمّيه `.env` وحط فيه المعلومات:
   - TELEGRAM_TOKEN
   - APP_KEY
   - APP_SECRET

2. شغّل البوت بـ:
```
pip install -r requirements.txt
python bot.py
```

3. أو انشره على Koyeb:
   - اربط GitHub
   - Build: `pip install -r requirements.txt`
   - Run: `python bot.py`
   - ضيف المتغيرات من `.env`
